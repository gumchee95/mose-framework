## 🔧 Excel 電話號碼清洗與重複項標記
**日期：** 2026-04-14
**情境：** 處理包含馬來西亞與外國電話號碼的 Excel 檔案，進行格式化與數據校驗。
**最佳實踐：**
- **格式標準化**：馬來西亞號碼統一使用 `60` 開頭（不帶 `+`）。若原始數據以 `01` 開頭則替換 `0` 為 `60`；若以 `1` 開頭則補齊 `60`。
- **符號清理**：使用 `re.sub(r'\D', '', s)` 移除所有非數字字符（空格、連字符、括號等）。
- **兩階段處理邏輯**：
  - 第一階段：清理號碼並收集到清單中。
  - 第二階段：使用 `Counter` 統計頻次，識別在整個文件中出現多次的重複號碼。
- **異常值標記 (Remarks)**：增加備註列，標記「長度異常 (Invalid Length)」、「重複 (Duplicate)」或「查無數字 (No Digits Found)」，以便人工複核。
- **多列處理**：同時處理本人與監護人號碼，並分別記錄備註。
